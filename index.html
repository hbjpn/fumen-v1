Welcome to Fumen Pages.

"Fumen" is a chord and rythm score rendering engine which run on the web browsers. Fumen is a pure client-side javascript program.

About Fumen

Fumen is a kind of "Markdown" text text processor. Fumen load text data markup with fumen grammer and generate score as SVG image on the web browser.

Creating pages manually

Markdown

Markdown Description
| Border of a measure
|| Border of a measure(Double border)
||: :|| Start/End of a loop
||:x4 End of a loop with number of loop
||. End of a piece
[A] Reharsal mark

Markdown in the mesure

Markdown Description
<S> Segno
<Coda> Coda
<Coda2> Coda with number. Number also can be specified with D.S. al Coda.
<D.S.> Dal Segno
<D.S.2> Dal Segno with number(any number can be specified)
<D.S. al Coda> Dal Segno al Segno
<D.C.> Da Capo
[1.] Loop number
(3/4) Time mark

Variables

Variable Value Example Description
TITLE Any string %TITLE="Thriller" Title of a piece
ARTIST Any string %ARTIST="Michael Jackson" Artist of a piece
TRANSPOSE Number %TRANSPOSE="+2" Transpose
